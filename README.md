# basic-ALP-programs-8051-bundle
A bundle of basic ALP programs for 8051. Covers move operations, addition, subtraction, multiplication, and division. Perfect for beginners. 
You may use any established simulator for 8051 such as EdSim51 or Keil Microvision 5. The software used for all code in this repo is EdSim51.
Hope it helps! Cheers
